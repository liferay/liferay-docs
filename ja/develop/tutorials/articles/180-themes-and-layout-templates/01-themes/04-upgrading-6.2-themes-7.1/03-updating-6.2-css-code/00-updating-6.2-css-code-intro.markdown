@product-ver@のUIの改善には、これらのCSS関連の変更が必要でした。

  - CSSファイルの名前変更
  - 不要なCSSファイルを削除する
  - CSSルールとインポートの更新
  - CSS応答性トークンの変更

テーマのアップグレードプロセスには、これらの変更への準拠が含まれます。 スタイルの改善を活用するためにテーマのCSSを更新する方法は次のとおりです。
